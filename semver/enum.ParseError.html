pub enum ParseError {
    NonAsciiIdentifier,
    IncorrectParse(Version, String),
    GenericFailure,
}

A ParseError is returned as the Err side of a Result when a version is attempted to be parsed.

Variants

NonAsciiIdentifier

All identifiers must be ASCII.

IncorrectParse

The version was mis-parsed.

GenericFailure

Any other failure.
